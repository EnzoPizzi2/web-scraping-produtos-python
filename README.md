# Web Scraping de Produtos com Python

Projeto em Python para extrair nome e preço de produtos de uma estrutura HTML e salvar os dados em uma planilha Excel.

## Funcionalidades

- leitura de estrutura HTML
- extração de dados de produtos
- criação de tabela com pandas
- exportação automática para Excel

## Tecnologias utilizadas

Python  
BeautifulSoup  
Pandas  
Openpyxl  

## Como executar

Instale as dependências:

py -m pip install -r requirements.txt

Depois execute:

py web_scraping_produtos.py

## Resultado

O script gera automaticamente o arquivo:

produtos.xlsx
